The Code for FIZZ BUZZ

            
//Get the starting and ending values from the page
//CONTROLLER FUNCTION
    function getValues(){
        let startValue = document.getElementById("startValue").value;
        let endValue = document.getElementById("endValue").value;
    

        //Parse into Intergers
            startValue = parseInt(startValue);
            endValue = parseInt(endValue);

        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
             //we call generateNumbers
            let numbers = generateNumbers(startValue, endValue);
            //we call displayNumbers
            displayNumbers(numbers);

        } else {
            alert("You must enter numbers")
        }
}

//Generate Numbers from the start start value to the end value
//LOGIC FUNCTION

    function generateNumbers(startValue, endValue){
        let numbers = [];

    for (let index = startValue; index <= endValue; index++) {
        numbers.push(index);
        
        }

    return numbers;
}

//Display the numbers from the starting to ending values 
//Change multiples of 3 to "Fizz"
//Change multiples of 5 to "Buzz"
//Change multiples of both to "FizzBuzz"
//VIEW FUNCTION

    function displayNumbers(numbers){

        let templateRows = "";
        let currentRow = "";
    for (let index = 0; index < numbers.length; index++) { 
    
        let className = "";
        let number = numbers[index];
        
        if (number % 3 == 0 && number % 5 == 0 ) {
            number = "FizzBuzz";
            className = "fizzbuzz";
        } else if (number % 3 == 0 ) {
           number = "Fizz";
           className = "fizz";
        } if (number % 5 == 0 ) {
            number = "Buzz";
            className = "buzz";
        }

        if (index % 5 == 0) {
            currentRow += "<tr>" //starting a new row
            templateRows += currentRow; //adding table data to the row
            currentRow = "</tr>"; //ending the current row
        }

         currentRow += `<td>< class=" ${className}">${number}</td>`; //adding data to the table
    }

        document.getElementById("results").innerHTML = templateRows;
}

A breakdown of my Functions

getValues

getValues is a function that receives the input from the page. It uses the getElementById to pull the values from the page. It then passes the values to the generateNumbers function which returns an array of values and passes the array to the displayNumbers function.

generateNumbers

generateNumbers takes in two parameters. The startValue and endValue inputted by the user. A variable is then created that holds an array. A for loop is used to generate all of the numbers between the startValue and the endValue.

displayNumbers

displayNumbers takes in the parameters set by the previous functions. The function then displays the numbers selected by the users startValue and endValue. I have created classes in CSS that are being passed through to the function through the className variable which instructs the multiples of three to display in green, the multiples of five to display in blue and the multiples of both three and five to display in red.

A for loop is used to check which numbers are multiples of three, which are multiples of five and which are multiples of both. I am using the (%) remainder operator to see if a number can be divided by three, five or both. If a number can be divided by three and five and returns a remainder of 0 then the number is changed to Fizzbuzz, the color is changed to red and displayed in bold. If it is divisable by three (else if) and returns a remainder of 0 then the number is changed to Fizz and displayed in green. If the number is divisable by five (if) and returns a remainder of 0 then the number is changed to Buzz and is displayed in blue. The loop is run through the templateRows variable and has the className introduced through the HTML row used for display until the endValue has been met. I have also added in a function to display the content in a table format through starting and ending a new row with currentRow and adding in the table data with templateRow.
